# Блок Задача / Результат
mission:
  task: "Забезпечити об'єкт із високим енергоспоживанням (понад 19 000 кВт⋅год/рік) абсолютною автономністю під час відключень мережі. Додаткова вимога — суттєво зменшити рахунки за електроенергію, використавши всі доступні площі для панелей."
  result: "Для покриття таких високих потреб інженери А100 змонтували потужну серверну стійку акумуляторів на 64 кВт⋅год та розробили комбіноване сонячне поле (частина панелей на даху, частина — на спеціальній наземній конструкції). Система повністю виправдала інвестиції: вона замістила 62% річного споживання будинку (згенерувавши понад 11 800 кВт⋅год власної енергії) та гарантує безперебійне живлення всього маєтку навіть при тривалих блекаутах."
