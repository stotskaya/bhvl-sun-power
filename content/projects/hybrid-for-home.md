# Блок Задача / Результат
mission:
  task: "Побудувати потужну станцію з акцентом на ефективну роботу в осінньо-зимовий період. Головною вимогою було не використовувати дах будинку, а знайти альтернативне рішення для розміщення панелей на ділянці."
  result: " Інженери А100 спроєктували та встановили посилене сонячне поле (14,1 кВт) на спеціальній наземній металоконструкції. Це дозволило не чіпати дах і виставити ідеальний кут нахилу модулів для захоплення низького зимового сонця.Результат перевершив очікування: незважаючи на запуск станції в серпні (напередодні холодного сезону), вона згенерувала 2240 кВт⋅год чистої енергії. Це дозволило покрити майже 80% (78,7%) усіх потреб будинку та забезпечити швидкий заряд преміальної батареї для захисту від блекаутів."
